How to use Translate Buddy

Please excuse this terrible interface .,. Tis' only an info screen

To begin: highlight text, right click, and press translate from the right click menu!

Shortcuts
When text is highlighted, the translate shortcut (for windows) is CTRL + SHIFT + ALT or (for apple devices) CMD + SHIFT + OPTION To access your word bank, the shortcut is CTRL + SHIFT + ALT + D on windows devices (this takes precedence over the above translate shortcut)
Other things to note:
The extension unloads dictionary data when you haven't used it in a short period of time (~half a minute) to save computation resources. When this happens, the next time a translation is requested, the extension may need a few seconds to load back the data.
